Omege UOS 文档中心

应用开发全流程：编写 → 编译 ELF → 集成到 ISO 镜像

1. 项目简介

Omege UOS 是一款轻量级、高性能的自研操作系统，原生支持 ELF 格式应用程序。

本教程将指导你完成：

  • 获取官方源码
  • 开发原生 APP
  • 编译为 ELF 可执行文件
  • 将 ELF 集成到 ISO 镜像
  • 编译并启动 Omege UOS ISO
官方仓库：https://github.com/deeplearning-corporation/OmegeUOS.git

2. 克隆官方源码

使用 git 直接克隆 Omege UOS 完整项目：

git clone https://github.com/deeplearning-corporation/OmegeUOS.git
cd OmegeUOS

克隆完成后，你将获得：

  • 系统内核
  • 根文件系统 rootfs
  • ISO 编译脚本
  • 应用开发模板

3. 搭建开发环境

3.1 安装依赖

# Ubuntu/Debian
sudo apt install gcc make binutils xorriso grub-pc-bin qemu-system-x86

3.2 验证编译工具

gcc --version
make --version

4. 编写 Omege UOS 原生 APP

Omege UOS 应用使用 C 语言开发，编译为 32/64 位 ELF 文件。

示例：hello.c

#include <stdio.h>

int main() {
  printf("Hello, Omege UOS!\n");
  return 0;
}

5. 编译为 ELF 文件

使用 GCC 直接编译为 Omege UOS 可执行的 ELF 文件：

gcc -m32 -o hello hello.c -nostdlib -fno-builtin

查看文件格式：

file hello

输出应为：ELF 32-bit LSB executable

6. 将 ELF 放入 ISO 镜像

把编译好的 ELF 应用复制到系统 rootfs 目录：

cp hello ./iso/rootfs/usr/bin/

路径：iso/rootfs/usr/bin/ 系统启动后可直接运行

7. 编译生成 Omege UOS ISO

运行官方构建脚本生成 ISO：

chmod +x build.sh
./build.sh

构建完成后会输出：omegeuos.iso

8. 运行 ISO 并测试 APP

qemu-system-x86_64 -cdrom omegeuos.iso -m 512

系统启动后，直接输入应用名运行：

hello

即可看到输出：Hello, Omege UOS!
